Política de Privacidad

Última actualización: 3 de julio de 2026

En esta web se respetan y protegen los datos personales de los usuarios. A continuación se explica qué información puede tratarse y con qué finalidad.

1. Responsable del tratamiento

Titular: Luis Guacache
Email: hola@luisguacache.com
Ubicación: Madrid, España

2. Datos que se recopilan

La web no incluye formularios propios. Los datos personales se facilitan únicamente cuando el usuario contacta voluntariamente por email, WhatsApp u otros canales enlazados desde la página.

Si el usuario acepta las cookies de análisis, la web puede utilizar Google Analytics para obtener estadísticas agregadas de navegación, como páginas visitadas, tipo de dispositivo, navegador, país aproximado o interacciones básicas. Estas métricas se usan para mejorar el sitio web.

3. Finalidad del tratamiento

Los datos de contacto se utilizan para responder consultas, preparar propuestas, coordinar servicios o mantener la comunicación solicitada por el usuario. Los datos estadísticos se utilizan para analizar el rendimiento de la web y mejorar su contenido.

4. Legitimación

La base legal para responder comunicaciones es el consentimiento del usuario al contactar de forma voluntaria. La base legal para las cookies de análisis es el consentimiento otorgado mediante el aviso de cookies.

5. Conservación de datos

Los datos se conservarán durante el tiempo necesario para atender la comunicación, gestionar la relación solicitada o cumplir obligaciones legales aplicables.

6. Servicios de terceros

Esta web puede enlazar a servicios externos como WhatsApp, Instagram o Google Analytics. Cada servicio puede aplicar sus propias políticas de privacidad y condiciones de uso.

7. Derechos del usuario

El usuario puede solicitar acceso, rectificación o eliminación de sus datos escribiendo a hola@luisguacache.com.
